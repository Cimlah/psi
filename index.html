GitHub Pages

GitHub Pages jest darmowym serwisem do hostowania stron. Sam GitHub jest platformą do publikacji oprogramowania na licencji typu open source. Dzięki temu, w bardzo prosty sposób można zacząć hostować stronę z kodu źródłowego swojego repozytorium. Przez swoją prostotę, jest to też serwis dający dosyć mało możliwości, bo pozwala tylko na hostowanie stron staycznych, nie posiada baz danych

Heroku

Popularny serwis wśród web-developerów posiadający bardzo elastyczne plany płatności. W podstawowym pakiecie można hostować stronę statyczną albo nawet cały backend. Jeżeli chce się dodać bazę danych lub np. większą ilość pamięci RAM na serwerze, trzeba zapłacić więcej.
